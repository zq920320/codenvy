Discover Codenvy in 5 Steps

A project has been created with an editor, cloned source code, builder, runner and plug-ins.


1
Run the App
Click Play.



A Docker microservice is launched, dependencies are auto-installed and the project is executed. Create custom environments through Docker recipes.
2
Use the App
The app’s URL will appear in the Runner panel, click to open your app in another tab.



Stop the app when you’re done.


3
Edit the App
In the Project Explorer, navigate to src > main > resources > db > hsqldb > populateDB.sql.

Replace one of the names with your own name. Save the file.
4
Commit the Changes
Select Git > Commit. Select the checkbox, add a comment and commit. You can’t push your change because this factory is not set up with a fork.

5
Persist and Share
Persist the project.



Persisting commits code changes, clones the artifacts, and copies builders and runners to your own workspace.

Share the project with the URL or by inviting someone into your persisted project.
